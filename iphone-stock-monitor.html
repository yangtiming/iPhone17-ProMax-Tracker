ℹ️ 使用说明 / Instructions

🔧 如何解决跨域问题（CORS）

  1. 安装浏览器扩展
    • Chrome用户：搜索并安装 "Moesif CORS" 或 "Allow CORS" 扩展
    • Firefox用户：搜索并安装 "CORS Everywhere" 扩展
    • Edge用户：搜索并安装 "Allow CORS" 扩展
  2. 启用扩展：点击浏览器工具栏的扩展图标，开启CORS功能
  3. 开始监控：选择你想监控的iPhone型号，点击"开始监控"按钮
  4. 注意：监控完成后，记得关闭CORS扩展以保证浏览安全

  3. 创建邮件模板，包含变量：{{product_name}}, {{store_info}}, {{time}}
  4. 获取Service ID、Template ID和Public Key
  5. 填入下方配置区域即可启用邮件提醒

🔧 How to Fix CORS Issues

  1. Install Browser Extension
    • Chrome: Search and install "Moesif CORS" or "Allow CORS" extension
    • Firefox: Search and install "CORS Everywhere" extension
    • Edge: Search and install "Allow CORS" extension
  2. Enable Extension: Click the extension icon in browser toolbar to enable CORS
  3. Start Monitoring: Select iPhone models you want to monitor and click "Start Monitoring"
  4. Note: Remember to disable CORS extension after monitoring for browsing safety

📧 Email Alert Setup (Optional)

  1. Visit EmailJS to create a free account
  2. Create an email service (choose Gmail or others)
  3. Create email template with variables: {{product_name}}, {{store_info}}, {{time}}
  4. Get your Service ID, Template ID and Public Key
  5. Enter them in the configuration below to enable email alerts
